# RetroTuner UI

Volumio plugin to control playback via a 16x2 VFD/LCD display, rotary encoder, and resistor ladder buttons wired to an MCP3008 ADC.

Originally built for a repurposed TEAC T-H300DAB MKII front panel, but works with any compatible hardware combination.

## Install

Copy the `retrotuner-ui` folder to `/tmp/retrotuner-ui`, then:

```bash
cd /tmp/retrotuner-ui
volumio plugin install
```
